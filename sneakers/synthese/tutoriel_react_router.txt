TUTORIEL REACT ROUTER - TRANSFORMATION SPA
===========================================

INTRODUCTION
============

Ce tutoriel montre comment transformer une application React d'une page unique vers une Single Page Application (SPA) avec React Router. Nous partons d'une application où tout était affiché sur une seule page et nous la transformons en plusieurs pages navigables.

ÉTAT INITIAL : APPLICATION MONOLITHIQUE
======================================

Notre application initiale dans App.jsx ressemblait à ceci :

```jsx
import {useState, useEffect} from 'react';
import Banner from './Banner';
import ShoppingList from './ShoppingList'
import Cart from './Cart';
import '../styles/App.css';

function App() {
    const [cart, setCart] = useState(() => {
        const savedCart = localStorage.getItem('cart');
        return savedCart ? JSON.parse(savedCart) : [];
    });

    useEffect(() => {
        localStorage.setItem('cart', JSON.stringify(cart));
    }, [cart]);

    const addToCart = (sneaker) => {
        // ... logique d'ajout
    };

    const removeFromCart = (sneaker) => {
        // ... logique de suppression
    }

    const clearCart = () => {
        setCart([]);
    }

    return (
        <div className="App">
            <Banner />
            <div className="main-content">
                <div className="shopping-section">
                    <ShoppingList onAddToCart={addToCart} />
                </div>
                <div className="cart-section">
                    <Cart
                        cartItems={cart}
                        onRemoveFromCart={removeFromCart}
                        onClearCart={clearCart}
                    />
                </div>
            </div>
        </div>
    );
}
```

PROBLÈMES DE CETTE APPROCHE :
- Tout sur une seule page
- Impossible de naviguer vers des sections spécifiques
- URLs non descriptives
- Difficile de partager des liens vers le panier
- Code monolithique difficile à maintenir

PARTIE 1 : ANALYSE ET PLANIFICATION
===================================

OBJECTIFS DE LA TRANSFORMATION :
- Séparer en pages distinctes : Accueil, Panier, Commande
- Ajouter la navigation entre pages
- Conserver la fonctionnalité existante
- Améliorer l'expérience utilisateur

STRUCTURE CIBLE :
src/
  Components/     ← Composants réutilisables
    Banner.jsx
    ShoppingList.jsx
    Cart.jsx
    App.jsx        ← Devient le routeur principal
  Pages/          ← Pages spécifiques
    HomePage.jsx   ← Liste des sneakers
    CartPage.jsx   ← Page dédiée au panier
    CheckoutPage.jsx ← Page de commande

PARTIE 3 : CRÉATION DES PAGES
=============================

3.1 HOMEPAGE.JSX - PAGE D'ACCUEIL
---------------------------------

Cette page reprend la fonctionnalité de la liste des sneakers :

```jsx
import React, { useState, useEffect } from 'react';
import Banner from '../Components/Banner';
import ShoppingList from '../Components/ShoppingList';

function HomePage() {
    const [cart, setCart] = useState(() => {
        const savedCart = localStorage.getItem('cart');
        return savedCart ? JSON.parse(savedCart) : [];
    });

    useEffect(() => {
        localStorage.setItem('cart', JSON.stringify(cart));
    }, [cart]);

    const addToCart = (sneaker) => {
        setCart(prevCart => {
            const existingItem = prevCart.find(item => item.id === sneaker.id);
            if (existingItem) {
                return prevCart.map(item =>
                    item.id === sneaker.id
                        ? { ...item, quantity: item.quantity + 1 }
                        : item
                );
            } else {
                return [...prevCart, { ...sneaker, quantity: 1 }];
            }
        });
    };

    const cartItemsCount = cart.reduce((total, item) => total + item.quantity, 0);

    return (
        <div>
            <Banner cartItemsCount={cartItemsCount} />
            <div style={{ padding: '20px', maxWidth: '1200px', margin: '0 auto' }}>
                <ShoppingList onAddToCart={addToCart} />
            </div>
        </div>
    );
}

export default HomePage;
```

POINTS CLÉS :
- Récupère la gestion d'état du panier depuis l'ancien App.jsx
- Inclut la fonction addToCart
- Calcule cartItemsCount pour l'afficher dans Banner
- Layout simplifié sans la section cart

3.2 CARTPAGE.JSX - PAGE DU PANIER
---------------------------------

Page dédiée exclusivement au panier :

```jsx
import React, { useState, useEffect } from 'react';
import Banner from '../Components/Banner';
import Cart from '../Components/Cart';
import { Link } from 'react-router-dom';

function CartPage() {
    const [cart, setCart] = useState(() => {
        const savedCart = localStorage.getItem('cart');
        return savedCart ? JSON.parse(savedCart) : [];
    });

    useEffect(() => {
        localStorage.setItem('cart', JSON.stringify(cart));
    }, [cart]);

    const removeFromCart = (sneaker) => {
        setCart(prevCart => {
            const existingItem = prevCart.find(item => item.id === sneaker.id);
            if (existingItem) {
                if (existingItem.quantity === 1) {
                    return prevCart.filter(item => item.id !== sneaker.id);
                } else {
                    return prevCart.map(item =>
                        item.id === sneaker.id
                            ? { ...item, quantity: item.quantity - 1 }
                            : item
                    );
                }
            }
            return prevCart;
        });
    };

    const clearCart = () => {
        setCart([]);
    };

    const cartItemsCount = cart.reduce((total, item) => total + item.quantity, 0);

    return (
        <div>
            <Banner cartItemsCount={cartItemsCount} />
            <div style={{ padding: '20px', maxWidth: '800px', margin: '0 auto' }}>
                <Cart
                    cartItems={cart}
                    onRemoveFromCart={removeFromCart}
                    onClearCart={clearCart}
                />
                {cart.length > 0 && (
                    <div style={{ marginTop: '20px', textAlign: 'center' }}>
                        <Link to="/checkout">
                            <button style={{ 
                                backgroundColor: '#4CAF50', 
                                color: 'white', 
                                padding: '10px 20px', 
                                border: 'none', 
                                borderRadius: '5px',
                                cursor: 'pointer'
                            }}>
                                Passer la commande
                            </button>
                        </Link>
                    </div>
                )}
            </div>
        </div>
    );
}

export default CartPage;
```

NOUVEAUTÉS :
- Bouton "Passer la commande" avec Link vers /checkout
- Page centrée sur le panier uniquement
- Layout optimisé pour afficher le panier

3.3 CHECKOUTPAGE.JSX - PAGE DE COMMANDE
---------------------------------------

Page basique pour la commande :

```jsx
import React, { useState, useEffect } from 'react';
import Banner from '../Components/Banner';

function CheckoutPage() {
    const [cart, setCart] = useState(() => {
        const savedCart = localStorage.getItem('cart');
        return savedCart ? JSON.parse(savedCart) : [];
    });

    const cartItemsCount = cart.reduce((total, item) => total + item.quantity, 0);
    const totalPrice = cart.reduce((total, item) => total + (item.price * item.quantity), 0);

    return (
        <div>
            <Banner cartItemsCount={cartItemsCount} />
            <div style={{ padding: '20px', maxWidth: '600px', margin: '0 auto' }}>
                <h2>Commande</h2>
                <p>Total : {totalPrice}€</p>
                <p>Page de commande - À développer...</p>
            </div>
        </div>
    );
}

export default CheckoutPage;
```

PARTIE 4 : TRANSFORMATION D'APP.JSX
===================================

4.1 NOUVEAU CODE APP.JSX
------------------------

Remplacer COMPLÈTEMENT le contenu d'App.jsx :

```jsx
import React from 'react';
import { BrowserRouter, Routes, Route } from 'react-router-dom';
import HomePage from '../Pages/HomePage';
import CartPage from '../Pages/CartPage';
import CheckoutPage from '../Pages/CheckoutPage';
import '../styles/App.css';

function App() {
    return (
        <BrowserRouter>
            <div className="App">
                <Routes>
                    <Route path="/" element={<HomePage />} />
                    <Route path="/cart" element={<CartPage />} />
                    <Route path="/checkout" element={<CheckoutPage />} />
                </Routes>
            </div>
        </BrowserRouter>
    );
}

export default App;
```

4.2 CE QUI A CHANGÉ
-------------------

SUPPRIMÉ de App.jsx :
- Tous les useState pour la gestion du panier
- Toutes les fonctions addToCart, removeFromCart, clearCart
- Tout le JSX complexe avec main-content, shopping-section, cart-section
- Les imports de ShoppingList, Cart, Banner

AJOUTÉ à App.jsx :
- Import de BrowserRouter, Routes, Route
- Import des nouvelles pages
- Structure de routing simple
- BrowserRouter qui enveloppe toute l'application

PARTIE 5 : MISE À JOUR DU BANNER
================================

5.1 BANNER AVANT (sans navigation)
----------------------------------

```jsx
function Banner() {
    return (
        <div className="banner">
            <h1>🏀 Jungle Sneakers</h1>
        </div>
    );
}
```

5.2 BANNER APRÈS (avec navigation)
----------------------------------

```jsx
import React from 'react';
import { Link } from 'react-router-dom';

function Banner({ cartItemsCount }) {
    return (
        <div className="banner">
            <h1>🏀 Jungle Sneakers</h1>
            <nav>
                <Link to="/">Accueil</Link>
                <Link to="/cart">
                    Panier ({cartItemsCount || 0})
                </Link>
            </nav>
        </div>
    );
}

export default Banner;
```

5.3 CHANGEMENTS DANS BANNER
---------------------------

AJOUTÉ :
- Import { Link } from 'react-router-dom'
- Prop cartItemsCount pour afficher le nombre d'articles
- Section <nav> avec liens de navigation
- Link vers "/" (Accueil)
- Link vers "/cart" (Panier avec compteur)
