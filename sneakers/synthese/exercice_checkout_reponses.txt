Question 1.2 - Données nécessaires
La CheckoutPage doit afficher un récapitulatif détaillé. De quelles informations a-t-elle besoin ?

REPONSES - Props nécessaires depuis App.jsx :
☑️ cart (tableau des articles) - NECESSAIRE pour afficher les articles et calculer les totaux
☑️ cartItemsCount (nombre total d'articles) - NECESSAIRE pour afficher le nombre d'articles
❌ addToCart (fonction d'ajout) - PAS NECESSAIRE sur la page checkout
❌ removeFromCart (fonction de suppression) - PAS NECESSAIRE sur la page checkout
❌ clearCart (fonction de vidage) - PAS NECESSAIRE sur la page checkout

JUSTIFICATION :
CheckoutPage affiche seulement un récapitulatif en lecture seule. Elle a besoin des données 
du panier (cart) pour les calculs et l'affichage, et du nombre d'articles (cartItemsCount) 
pour les informations. Les fonctions de modification ne sont pas nécessaires car cette page 
ne permet pas de modifier le panier.

CODES COMPLETS - SOLUTIONS

CHECKOUTPAGE.JSX - SOLUTION COMPLETE
------------------------------------
```jsx
import { Link } from 'react-router-dom';
import '../styles/CheckoutPage.css';

function CheckoutPage({ cart, cartItemsCount }) {
    // Calculs financiers
    const sousTotal = cart.reduce((total, item) => total + (item.prix * item.quantity), 0);
    const tva = sousTotal * 0.20;
    const totalTTC = sousTotal + tva;
    
    // Si panier vide, afficher message
    if (cartItemsCount === 0) {
        return (
            <div className="checkout-page">
                <h2>Finaliser votre commande</h2>
                <p>Votre panier est vide.</p>
                <Link to="/" className="btn-secondary">Retourner au catalogue</Link>
            </div>
        );
    }
    
    return (
        <div className="checkout-page">
            <h2>Finaliser votre commande</h2>
            
            {/* Nombre d'articles */}
            <div className="order-info">
                <p>Nombre d'articles : <strong>{cartItemsCount}</strong></p>
            </div>
            
            {/* Liste des articles */}
            <div className="order-summary">
                <h3>Récapitulatif de votre commande</h3>
                {cart.map(item => (
                    <div key={item.id} className="order-line">
                        <span className="item-info">{item.nom} x{item.quantity}</span>
                        <span className="item-price">{(item.prix * item.quantity).toFixed(2)}€</span>
                    </div>
                ))}
            </div>
            
            {/* Totaux */}
            <div className="order-totals">
                <div className="total-line">
                    <span>Sous-total HT:</span>
                    <span>{sousTotal.toFixed(2)}€</span>
                </div>
                <div className="total-line">
                    <span>TVA (20%):</span>
                    <span>{tva.toFixed(2)}€</span>
                </div>
                <div className="total-line total-final">
                    <strong>
                        <span>Total TTC:</span>
                        <span>{totalTTC.toFixed(2)}€</span>
                    </strong>
                </div>
            </div>
            
            {/* Navigation */}
            <div className="checkout-actions">
                <Link to="/cart" className="btn-secondary">← Retour au panier</Link>
                <Link to="/shipping" className="btn-primary">
                    Continuer vers la livraison →
                </Link>
            </div>
        </div>
    );
}

export default CheckoutPage;
```
