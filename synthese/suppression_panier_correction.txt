CORRECTION : GESTION DE LA SUPPRESSION D'ARTICLES DU PANIER
============================================================

📋 RÉPONSES AUX QUESTIONS GUIDÉES
==================================

🔍 ÉTAPE 1 : ANALYSER L'ARCHITECTURE EXISTANTE
-----------------------------------------------

✅ Q1 : Quel composant gère actuellement l'état du panier ?
RÉPONSE : App.jsx
EXPLICATION : C'est dans App.jsx que se trouve le useState([]) pour initialiser le panier. 
Le composant parent gère l'état global de l'application.

✅ Q2 : Comment fonctionne actuellement l'ajout d'un article au panier ?
RÉPONSE : 
1. Clic sur "Ajouter au panier" dans SneakerItem.jsx
2. Appel de onAddToCart(sneaker) - prop reçue du parent
3. Dans App.jsx : addToCart(sneaker) modifie l'état avec setCart()
4. React re-rend automatiquement tous les composants impactés

✅ Q3 : Quel composant affiche le contenu du panier ?
RÉPONSE : Cart.jsx
EXPLICATION : Ce composant reçoit cartItems en props et s'occupe de l'affichage.

🔍 ÉTAPE 2 : PLANIFIER LA SUPPRESSION D'UN ARTICLE
---------------------------------------------------

✅ Q4 : Dans quel composant devez-vous créer la fonction de suppression ?
RÉPONSE : b) Dans App.jsx (parent qui gère l'état)
EXPLICATION : Seul le composant qui possède l'état peut le modifier. 
Les composants enfants ne peuvent que demander des modifications via des callbacks.

✅ Q5 : Quels paramètres cette fonction doit-elle recevoir ?
RÉPONSE : L'article à supprimer (ou son id)
EXPLICATION : Pour identifier précisément quel article supprimer du panier.
Exemple : removeFromCart(itemToRemove)

✅ Q6 : Comment cette fonction doit-elle être transmise au composant Cart ?
RÉPONSE : Via les props, comme pour addToCart
EXEMPLE :
```jsx
<Cart
    cartItems={cart}
    removeFromCart={removeFromCart}
    clearCart={clearCart}
/>
```

✅ Q7 : Dans Cart.jsx, où devez-vous placer le bouton "Supprimer" ?
RÉPONSE : Dans la boucle .map(), pour chaque article affiché
EXEMPLE :
```jsx
{cartItems.map(item => (
    <div key={item.id} className="cart-item">
        {/* Affichage de l'article */}
        <button onClick={() => handleRemoveFromCart(item)}>
            Supprimer
        </button>
    </div>
))}
```

🔍 ÉTAPE 3 : GESTION DES ÉVÉNEMENTS
------------------------------------

✅ Q8 : Pourquoi avez-vous besoin d'une fonction "handle" dans Cart.jsx ?
RÉPONSE : b) Pour passer l'article spécifique à supprimer
EXPLICATION : La fonction handle permet de "capturer" l'article courant dans la boucle 
et de le transmettre à la fonction de suppression du parent.

✅ Q9 : Quelle est la différence entre ces deux écritures ?
RÉPONSE :
- onClick={onRemoveFromCart} → Passe l'événement automatiquement, mais pas d'article spécifique
- onClick={() => onRemoveFromCart(item)} → Passe l'article spécifique en paramètre
CONCLUSION : Il faut utiliser la deuxième forme pour passer l'article à supprimer.

🔍 ÉTAPE 4 : LOGIQUE DE SUPPRESSION
------------------------------------

✅ Q10 : Comment vérifiez-vous si un article existe dans le panier ?
RÉPONSE : Avec .find()
EXEMPLE :
```javascript
const existingItem = cart.find(item => item.id === itemToRemove.id);
```

✅ Q11 : Quelle méthode JavaScript utilisez-vous pour :
RÉPONSES :
- Supprimer complètement un article ? → .filter()
- Réduire la quantité d'un article ? → .map()

EXEMPLE COMPLET :
```javascript
const removeFromCart = (itemToRemove) => {
    const existingItem = cart.find(item => item.id === itemToRemove.id);
    
    if (existingItem.quantity === 1) {
        // Supprimer complètement l'article
        setCart(cart.filter(item => item.id !== itemToRemove.id));
    } else {
        // Réduire la quantité de 1
        setCart(cart.map(item => 
            item.id === itemToRemove.id 
                ? { ...item, quantity: item.quantity - 1 }
                : item
        ));
    }
};
```

✅ Q12 : Comment respectez-vous le principe d'immutabilité en React ?
RÉPONSE : b) Créer un nouveau tableau avec les modifications
EXPLICATION : On utilise .filter() et .map() qui créent de nouveaux tableaux.
On ne modifie JAMAIS directement le tableau existant.

🔍 ÉTAPE 5 : VIDER LE PANIER
-----------------------------

✅ Q13 : Pour vider le panier, quelle fonction simple pouvez-vous créer ?
RÉPONSE :
```javascript
const clearCart = () => {
    setCart([]);
};
```
EXPLICATION : Remettre le panier à un tableau vide, son état initial.

✅ Q14 : Avez-vous besoin d'une fonction "handle" pour vider le panier ?
RÉPONSE : NON
EXPLICATION : Pas de paramètres à passer, on peut directement faire :
```jsx
<button onClick={clearCart}>Vider le panier</button>
```

🔍 ÉTAPE 6 : INTERFACE UTILISATEUR
-----------------------------------

✅ Q15 : Où devez-vous placer le bouton "Vider le panier" ?
RÉPONSE : c) Dans le résumé du panier (en bas)
EXPLICATION : Logiquement après avoir vu tous les articles et le total.

✅ Q16 : Faut-il afficher le bouton "Vider le panier" quand le panier est vide ?
RÉPONSE : NON
EXPLICATION : Pas de sens UX. Affichage conditionnel :
```jsx
{cartItems.length > 0 && (
    <button onClick={clearCart}>Vider le panier</button>
)}
```

🚀 IMPLÉMENTATION COMPLÈTE - SOLUTION
======================================

📁 App.jsx - AJOUT DES FONCTIONS
---------------------------------
```javascript
// Fonction pour supprimer/réduire un article
const removeFromCart = (itemToRemove) => {
    const existingItem = cart.find(item => item.id === itemToRemove.id);
    
    if (existingItem.quantity === 1) {
        // Supprimer complètement si quantité = 1
        setCart(cart.filter(item => item.id !== itemToRemove.id));
    } else {
        // Réduire la quantité de 1 si quantité > 1
        setCart(cart.map(item => 
            item.id === itemToRemove.id 
                ? { ...item, quantity: item.quantity - 1 }
                : item
        ));
    }
};

// Fonction pour vider complètement le panier
const clearCart = () => {
    setCart([]);
};

// Transmission des props au composant Cart
<Cart
    cartItems={cart}
    removeFromCart={removeFromCart}
    clearCart={clearCart}
/>
```

📁 Cart.jsx - AJOUT DES BOUTONS ET HANDLERS
--------------------------------------------
```javascript
function Cart({ cartItems, removeFromCart, clearCart }) {
    // Handler pour la suppression d'un article spécifique
    const handleRemoveFromCart = (itemToRemove) => {
        removeFromCart(itemToRemove);
    }

    // Affichage conditionnel si panier vide
    if (cartItems.length === 0) {
        return (
            <div className="cart">
                <h2>Mon Panier</h2>
                <p>Votre panier est vide</p>
            </div>
        );
    }

    return (
        <div className="cart">
            <h2>Mon Panier ({totalItems})</h2>
            
            {/* Liste des articles avec bouton supprimer */}
            <div className="cart-items">
                {cartItems.map(item => (
                    <div key={item.id} className="cart-item">
                        <img src={item.image} alt={item.nom} />
                        <div className="cart-item-details">
                            <h4>{item.nom}</h4>
                            <p>{item.marque}</p>
                            <p>Quantité: {item.quantity}</p>
                            <p>{item.prix * item.quantity}€</p>
                        </div>
                        <div className="cart-item-actions">
                            <button 
                                onClick={() => handleRemoveFromCart(item)}
                                className="remove-btn"
                            >
                                Supprimer
                            </button>
                        </div>
                    </div>
                ))}
            </div>
            
            {/* Résumé */}
            <div className="cart-summary">
                <h3>Total: {totalPrice}€</h3>
            </div>
            
            {/* Bouton vider le panier - affiché seulement si articles présents */}
            {cartItems.length > 0 && (
                <div className="cart-actions">
                    <button onClick={clearCart} className="clear-cart-btn">
                        Vider le panier
                    </button>
                </div>
            )}
        </div>
    );
}
```
